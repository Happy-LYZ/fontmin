# fontmin
字体压缩Demo

# 操作步骤

## 1、新建文件夹fontmin，并在此文件夹下执行如下命令：
```shell
npm install -g fontmin
```
## 2、创建如下文件目录：
-  2.1、“fontmin\asset\font”；        ---压缩后的字体文件目录
-  2.2、“fontmin\src\font\xxx.ttf”；  ---源字体文件目录（此目录下存放源字体文件"xxx.ttf"）
-  2.3、“fontmin\index.js”;           ---执行脚本文件
